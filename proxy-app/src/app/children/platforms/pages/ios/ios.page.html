Установка клиента V2BOX для iOS

V2BOX - клиент, с помощью которого можно будет подклбчиться к нашему прокси-серверу. Для работы используется VLESS протокол.

1. Скачать приложение из AppStore
2. Запросить ссылку на подключение в телеграм
3. Открыть приложение, перйти в раздел "Configs", и добавить подписку "Add Subscription"
4. Придумайте название для вашей подписки, вставьте ссылку из пункта 2 и нажмите на "Add Subscribe"
Увидели алерт "Subscribe added successfully" - идем дальше, нет - напишите в телеграм
5. Жмем на "VLESS" (выбираем настройки)
6. Переходим на главную "Home"
7. Скопировать настройки:
8. Переходим в "Routing" и нажимаем Enable routing, сверху нажмите вставить
9. Перезапускаем приложение
10. На главной нажмите "Slide to Connect"
Если хотите, чтобы клиент работал, только когда телефон включен - кликните на чекбокс "Stop on sleep".
